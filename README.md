### 新規アカウント登録までの手順：  

1. **トップページの「サインイン」を押下**  
   - ログイン画面へ遷移  
   - ログイン後ページ（ユーザーダッシュボード）へ  
   - 左上のロゴからトップページへ戻る  

2. **トップページの「お問い合わせ」を押下**  
   - データベース登録状況確認画面へ  
   - ※管理者アカウントの場合、編集・削除ボタンが出現  

3. **ログアウトボタンよりトップページに戻る**  
   - 「サインイン」を押下しログイン画面へ  
   - ログイン画面で「サインアップ」を押下  
   - サインアップ画面で ID と pw（任意の文字）を入力し、「サインイン」を押下
